# HR Analytics Dashboard using Excel

This repository contains the HR Analytics Dashboard developed using Excel.

## Objective

To create an interactive HR dashboard that provides insights into employee metrics such as attrition, job satisfaction, and departmental performance, enabling data-driven HR decisions.

## Project Q&A

1. What is the total employee population and how many are active vs attrited?  
2. What is the overall attrition rate and average age of employees?  
3. How does attrition vary by gender?  
4. How does attrition vary by education level?  
5. Which departments have the highest attrition rates?  
6. How does age impact attrition?  
7. How does marital status correlate with attrition?  
8. Which job roles have the highest attrition?
